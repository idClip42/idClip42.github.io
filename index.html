2015

Built in Unity. Models made in Blender3D

In this game, the player must survive against a large amount of zombies (up to 1000) in a neighborhood while searching for survival supplies.

The zombies use flocking algorithms to move towards the player, avoiding obstacles.

The zombies use sprites, rather than models, in order to allow for as many zombies as possible

All 3D models in this game were created by me, with the exception of the standard Unity trees.

Notes:
  • Climbing through windows often doesn't work in the web player version of the game, for some reason.
  • Turn on a radio.
  • Find the cabin.
  • Press Esc to unlock the mouse from the window.

Fonts Used: Glacial Indifference Font, Blade Runner Font
